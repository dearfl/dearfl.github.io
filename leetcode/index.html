leetcode

This page is an attempt to solve leetcode problems with Rust. The whole page is synced per day with my personal fork of leetcode-sync, note we only keep the last submission, and to simplify the process, I write the intuition as comment in submissions. (Currently we don’t show question id because leetcode-sync sometimes returns the wrong question id, however they should be correctly ordered by quesion id.)

I tries to write the solution in the most ergonomic Rust way, although sometimes leetcode doesn’t make it easy.
